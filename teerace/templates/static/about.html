What builds Teerace?

Teerace depends on many 3rd-party projects. Here's the complete list of them:

  • brabeion - badges, unleashed
  • celery - distributed task queue
  • django - PONIES <3
  • django-admin-tools - pretty admin site
  • django-annoying - a must-have for fixing django annoying things
  • django-countries - provides country list
  • django-crumbs - breadcrumbs solution
  • django-faq - simple FAQ app
  • django-piston - RESTful API framework
  • django-threadedcomments - we just got served
  • johnny-cache - low-level caching framework
  • M2Crypto - Python encryption module, Teerace uses it for encrypting connection with its gameservers
  • Markdown - ultimate markup language
  • memcached - cache backend
  • nginx - high-performance HTTP server
  • rabbitmq - AMQP messaging system, backend for celery
  • tml - Teeworlds map library, map data extraction and screenshot creation
  • uwsgi - application container server
